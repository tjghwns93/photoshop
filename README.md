# 포토샵기능 메모

```
웹디자인시험-산업인력공단(이론시험 + 실기)
ctrl + T --> 변형툴
alt + 마우스훨조정 --> 화면크기 조절
ctrl + 0 --> 화면에 맞추기
ctrl + 1 --> 화면 100%
ctrl + j --> 레이어복사
art + 드래그 --> 레이어복사
ctrl + z --> 실행취소
```
